# Restaurant Ratings Analysis Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard designed to analyze restaurant ratings data. The dashboard helps identify trends in customer ratings, pricing, cuisines, and location-based performance to support data-driven insights in the food and hospitality industry.

## Objectives
- Analyze restaurant ratings and performance
- Understand pricing and cost distribution
- Identify popular cuisines
- Compare restaurant performance across locations
- Practice Power BI data modeling and DAX

## Key Insights
- Ratings distribution across restaurants
- Average rating by cuisine
- Price range vs customer rating analysis
- Location-wise restaurant performance
- High and low performing restaurant categories

## Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated Measures)
- Data Visualization & Dashboard Design
